# Arcanum

**A single-file worldbuilding wiki for storytellers.**

> Live app: [Arcanum](https://supernova0866.github.io/Arcanum/)

Arcanum is a browser-based worldbuilding tool that runs entirely client-side. No server, no install — just open [Live Page](https://supernova0866.github.io/Arcanum/) and start building your world. Your data lives in your own Supabase or Turso database.

---

## Features

- **Worlds** — create multiple worlds, each with its own set of tabs and entries
- **Custom tabs** — define your own categories (Characters, Locations, Factions, etc.) with fully custom fields
- **Rich text editor** — formatting toolbar, headings, bold, italic, lists, and focus mode for distraction-free writing
- **@ entry linking** — type `@` in any rich text field to link to another entry
- **Entry status** — mark entries as Draft, Planned, In Progress, or Complete
- **Star / pin** — star important entries for quick access
- **Global search** — search across all worlds, tabs, and entries at once
- **8 themes** — Nebula, Ember, Abyss, Blood & Iron, Verdant, Parchment, Frost, Void
- **Custom icons** — choose emoji or upload your own icons for tabs
- **Export / Import** — download your world data as JSON, import it back any time
- **Card and table views** — switch between visual card grid and compact table layout
- **Duplicate entries** — clone any entry as a starting point
