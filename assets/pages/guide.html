SETUP GUIDE
How to connect Arcanum to your database
Step 01 Create a Supabase account

Go to supabase.com and sign up for a free account. No credit card required. Once signed in, you'll land on your dashboard.

🖼️ supabase-signup.png
Supabase homepage — click "Start your project"
Step 02 Create a new project

Click New Project, give it a name (e.g. arcanum), set a strong database password, and choose your region. Click Create new project and wait for it to provision — takes about 30 seconds.

🖼️ supabase-new-project.png
New project form — name, password, and region
⚠️ Save your database password somewhere safe — you won't be able to see it again.
Step 03 Get your Project URL and API key

Once the project is ready, go to Project Settings → API. Copy the Project URL and the publishable key — you'll need both to connect Arcanum.

🖼️ supabase-api-keys.png
Settings → API — copy the Project URL and publishable key
💡 Use the anon key, not the service_role key. The publishable key is safe to use in the browser.
Step 04 Connect in Arcanum

Open Arcanum, click the ⚙ Settings button, and go to the Database tab. Select Supabase as your provider, paste in your Project URL and publishable key, then click Connect. Arcanum will create the required tables automatically.

🖼️ arcanum-db-connect.png
Arcanum Settings → Database — paste your credentials
💡 Your credentials are stored in localStorage — they stay on your device and are never sent anywhere except directly to Supabase.
Step 01 Create a Turso account

Go to turso.tech and sign up. Turso has a generous free tier — 500 databases, 9GB storage. Sign in with GitHub for the quickest setup.

🖼️ turso-signup.png
Turso homepage — sign up with GitHub
Step 02 Create a database

From your Turso dashboard, click Create Database. Give it a name like arcanum and pick a region close to you. Click Create Database to confirm.

🖼️ turso-create-db.png
Turso dashboard — create a new database
Step 03 Get your Database URL and token

Open your database and go to Generate Token. Set expiry to No expiry and click Generate. Copy the token. Also copy the Database URL shown at the top — it looks like libsql://your-db.turso.io.

🖼️ turso-token.png
Turso database page — generate a token and copy the URL
⚠️ Copy the token immediately — Turso only shows it once.
Step 04 Connect in Arcanum

Open Arcanum, go to Settings → Database, select Turso, paste your Database URL and token, and click Connect. Tables are created automatically on first connect.

🖼️ arcanum-turso-connect.png
Arcanum Settings → Database — select Turso and paste credentials
💡 Turso works great if you're already using it for other projects — you can reuse the same account across multiple Arcanum instances with separate databases.
